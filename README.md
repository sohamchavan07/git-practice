# git-practiceA tiny sandbox to illustrate manual versioning vs Git.## Files- `story.txt`: base story text- `story_v1`, `story_v2`: manual copies simulating versions- `limitations.txt`: three drawbacks of manual versioning## Try Git hereInitialize a repo and capture history:```bashgit initgit add .git commit -m "Initial commit: add story, versions, limitations"```Make a change to `story.txt`, then record it:```bashgit add story.txtgit commit -m "Update story"```View history and differences:```bashgit log --onelinegit diff HEAD~1..HEAD```## Feature: README update on feature-readmeThis README was updated on the `feature-readme` branch to complete the assignment.
